Library Manager

Desktop application for managing a library system with admin and user roles.
Built using C#, WPF, MVVM, and SQL Server LocalDB.

Features
Login system (Admin / User)
Add books
Edit books
Delete books
Manage users (Admin)
Borrow books (User)
View borrowed books
Search books (by title and author)
Automatic quantity update when borrowing
Store data in SQL Server LocalDB
MVVM architecture
ICommand (RelayCommand implementation)
INotifyPropertyChanged
ObservableCollection data binding

Technologies
C#
WPF
.NET
SQL Server LocalDB
MVVM pattern

Future Improvements
Return book functionality
LINQ statistics 
Filters and sorting (OrderBy)
UI improvements (modern design)
